Banking Transaction Analysis

A MySQL project that simulates a basic banking system with customer accounts and transaction records.

## Database Structure
- **customers** — stores customer details, account type, and balance
- **transactions** — stores transaction records linked to customers via foreign key

## What's Covered
- Database and table creation
- INSERT with 10 customers and 10 transactions
- WHERE, ORDER BY, DISTINCT, LIMIT, LIKE (wildcard)
- GROUP BY, HAVING
- JOIN (customers ↔ transactions)
- Window Functions — Running Total (SUM OVER), DENSE_RANK
- CTE — average transaction comparison

## Tools Used
- MySQL Workbench
- MySQL 8.0

## Dataset
Dummy data with 10 Indian customers across cities like Hyderabad, Delhi, Mumbai, Chennai, and Bangalore.
